# Underwater Sound Samples

Sound samples for compression testing.  This a collection of audio samples for 
compression testing. Primarily for the (x3 audio compressor)[https://github.com/psiphi75/x3-spec].

The samples are referenced in this paper: 
[Low complexity lossless compression of underwater sound recordings](https://pubmed.ncbi.nlm.nih.gov/23464010/)

## Samples

These samples are sourced from https://soundtags.wp.st-andrews.ac.uk/dtags/audio_compression/audio-samples/
